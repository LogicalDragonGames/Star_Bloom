You can see the animations in the demo scene:
- By default you see "idle" animation
- If you keep "Space" pressed you see "attack" animation
- If you keep "UpArrow" pressed you see "run" animation

The mesh of the model's got 3000 triangles.
Depending on the material you choose, the number of tris increases
- mat_basic keep 3000 tris
- mat_lighted or mat_outliner increases the number of tris to 6000
- mat_high (lighted and outliner) increases the number of tris to 9000
